# SAR Demo - Frontier Gaming Systems

A standalone demo of the SAR (Standalone Analytical Reporting) application for bingo hall operations.

## Features

- **Dashboard View** - Overview of revenue, attendance, and key metrics
- **Event Browser** - Browse 92 events from January-November 2025
- **Charts** - 12-month revenue trends with 50% year-over-year growth
- **Master Sheet View** - Detailed spreadsheet-style data view
- **Notifications** - Alert system with clickable event navigation
- **Hotball Tracking** - Monitor progressive jackpot status

## Demo Data

This demo includes:
- 58 Mega Bingo Hall (MB) events
- 34 Small Bingo Hall (SB) events
- 50% revenue growth trajectory over the year
- Special events for demonstration:
  - Record Strip Sales event (10/18/2025)
  - Anomaly Alert event (11/29/2025)
